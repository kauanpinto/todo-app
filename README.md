# Todo App

Aplicação de gerenciamento de tarefas desenvolvida com foco em autenticação, persistência de dados e integração com Supabase.

## Funcionalidades

- Cadastro e login de usuários
- Recuperação e alteração de senha por e-mail
- Dashboard para gerenciamento de tarefas
- Criar, editar, excluir e marcar tarefas como concluídas

## Demonstração

- [Deploy da aplicação](https://todo-app-kauanpinto.vercel.app)
- [Repositório](https://github.com/kauanpinto/todo-app)

## Tecnologias

- React + TypeScript
- Tailwind CSS
- Supabase (Auth + Database)
- React Router
- Vite

## Variáveis de ambiente

Crie um arquivo `.env.local` na raiz do projeto e adicione as variáveis do seu projeto Supabase:

```bash
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

## Como executar

Clone o repositório:
```bash
git clone https://github.com/kauanpinto/todo-app.git
```

Instale as dependências e execute:
```bash
npm install && npm run dev
```
